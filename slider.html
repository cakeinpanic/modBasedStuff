About

Image rotator based on dfcreative's mod.
Rotation starts on mouse rolled over. Whem mouse moves out – slideshow returns to first image.
Standart functional is extended by "show me last slide" option with external manage (it is useful if you want to add some special last slide and let user stop slideshow on it)

How to use

<link rel="stylesheet" type="text/css" href="sliderStyle.css"> After DOM: <script src="mod.min.js" type="text/javascript"></script> <script src="slideshow.js" type="text/javascript"></script> Define your images array (images would be scaled to slider size), special last image (if needed), and seе preloader attribute to True if you want it to be showed. Otherway - don't set it ever. <div class="slider" slides="1.jpg,2.jpg,3.jpg" datamap="1_big.jpg" preloader="True" </div> To show special last slide – bind it from outside with method showLast(); <script type="text/javascript"> $(document).on("mouseenter", ".last-slide-btn", function() { $(e.target).closest('.slider').showLast(); }); $(document).on("mouseleave", ".last-slide-btn", function() { $(e.target).closest('.slider').resetToNormal(); }); </script>

Customization

Rotator is fully costomizable by css.
